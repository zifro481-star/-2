# Лидеры Права: сайт + админ-панель

Теперь сайт работает не только как статическая страница, но и как мини-система приёма заявок:

- публичная форма на главной странице отправляет заявку на сервер;
- данные сохраняются в `SQLite` базе `data/applications.db`;
- админ-панель доступна по адресу `/admin`;
- в админке можно просматривать заявки, менять статус и оставлять заметки.

## Как запустить

```bash
cd "/Users/ilya/Downloads/лидеры права сайт"
python3 server.py
```

После запуска:

- сайт: `http://127.0.0.1:8000/`
- админка: `http://127.0.0.1:8000/admin`

## Доступ в админку

По умолчанию:

- логин: `admin`
- пароль: `change-me-please`

Лучше сразу задать свой пароль:

```bash
ADMIN_PASSWORD="ваш-надежный-пароль" python3 server.py
```

При желании можно поменять и логин:

```bash
ADMIN_LOGIN="superadmin" ADMIN_PASSWORD="ваш-надежный-пароль" python3 server.py
```

## Где лежит база

Файл базы создаётся автоматически:

```text
data/applications.db
```
